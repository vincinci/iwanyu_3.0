## 🚀 How to Start

### Backend (Terminal 1)
```bash
cd backend
npm install
npm run db:generate
npm run dev
```

### Frontend (Terminal 2)
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Access Points
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
